# Glossary

## Session

A durable runtime conversation container spanning an operator, connector, automation task, or tracked dispatch execution.

## Run

A single live model execution inside a session.

## Dispatch

A durable Projects Engine record that requests one runtime execution.

## Thread

A durable tracked-work conversation/work log grouping jobs and dispatches.

## Worktree

A durable record for a repo execution context used by tracked work.

## Handoff

An idempotent submission from local or runtime-originated work into Aria Projects.

## Session Attachment

A paired-device binding to a live runtime session with scoped permissions.
