# Aria Specs

Canonical engineering specs for the Aria monorepo live here.

These specs are the package-level source of truth for:

- public product shape
- runtime execution behavior
- relay trust and remote attachment
- projects-engine durable work state
- security boundaries
- migration and cutover rules

`docs/` still contains operator and subsystem documentation. `specs/` captures the canonical engineering contracts that the implementation is expected to converge on.
